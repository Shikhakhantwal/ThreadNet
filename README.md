# 🚨 ThreatNet

### *A Smart Network Security Monitoring Tool for Real-Time Threat Detection*

**"Monitor. Detect. Defend — All in Real-Time."**
## ❓ Why ThreatNet?

In today’s digital ecosystem, **network threats are evolving faster than traditional defenses**. Most existing tools are either:

* Too complex for beginners
* Expensive or proprietary
* Lacking real-time visualization

### 💡 Solution

**ThreatNet** is designed as a lightweight, efficient, and educational **network security monitoring tool** that:

* Captures and analyzes live network traffic
* Detects suspicious patterns
* Provides real-time alerts and insights



## ✨ Key Features

* ⚡ **Real-Time Packet Analysis** — Monitor live network traffic instantly
* 🔍 **Threat Detection Engine** — Identify malicious patterns and anomalies
* 📊 **Threat Visualization** — Graphical/CLI representation of traffic data
* 🚨 **Alert System** — Instant notifications for suspicious activity
* 🧠 **Protocol Inspection** — Analyze TCP/IP headers and packet structure
* 🖥️ **Lightweight CLI Interface** — Fast and efficient execution
* 🔐 **Custom Rule Engine** — Define your own detection rules

## 🌐 Additional Features

* 🌍 **Multi-Language Support**
  Supports multiple languages for better accessibility and user experience:

  * English
  * हिन्दी (Hindi)
  * ગુજરાતી (Gujarati)

* 📝 **Feedback System (Admin-Protected)**

  * Users can submit feedback directly through the system
  * Admin authentication ensures only authorized access to feedback data
  * Helps in continuous improvement and monitoring user experience

* 🛠️ **Utility Tools**
  Built-in tools to enhance functionality beyond log analysis:

  * 🔐 Encode/Decode utilities (for basic encryption/decryption tasks)

  * 📊 Additional helper features for analysis and debugging
